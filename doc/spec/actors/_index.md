# Actors

This section defines the primary actors interacting with the system.
An **Actor** represents a role played by a user or an external system that interacts with the solution.

## Defining a New Actor

When adding a new actor, create a new file (e.g., `new-actor.md`) following this structure:

```markdown
<a id="ACT-{{ID}}"></a>

## ACT-{{ID}} {{NAME}}

{{SHORT_DESCRIPTION (2-3 lines)}}

**Roles** (Optional):

* **{{ROLE_NAME}}**: {{ROLE_DESCRIPTION}}

**Details** (Optional):

{{DETAILED_DESCRIPTION}}
```
