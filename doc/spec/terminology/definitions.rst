Terminology
===========

This section defines key terms and abbreviations used throughout the specification.

.. _TERM-SYS-CP:

Control Plane (TERM-SYS-CP)
---------------------------
The system described in this specification. It provides centralized management functions such as authentication, feature flagging, and auditing for multiple B2B SaaS applications.

.. _TERM-APP-TARGET:

Managed Application (TERM-APP-TARGET)
-------------------------------------
The B2B SaaS applications that are managed by the :ref:`TERM-SYS-CP`. These applications delegate authentication and feature control to the Control Plane.
Also referred to as "Target Application" or "B2B SaaS".
