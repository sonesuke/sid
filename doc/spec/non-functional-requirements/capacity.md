# Capacity

<a id="NFR-CAP-001"></a>

## NFR-CAP-001 Tenant Scalability

The system SHALL scale to support a minimum of 1,000 concurrent active [Tenants](../data-model/schema.md#DAT-TENANT).

**Constrained by**: [CON-CAP-001](../constraints-and-assumptions/index.md#CON-CAP-001)

<a id="NFR-CAP-002"></a>

## NFR-CAP-002 User Scalability

The system SHALL scale to support a minimum of 10,000 [Users](../data-model/schema.md#DAT-USER) per [Tenant](../data-model/schema.md#DAT-TENANT).

**Constrained by**: [CON-CAP-002](../constraints-and-assumptions/index.md#CON-CAP-002)
