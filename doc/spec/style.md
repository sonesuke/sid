# Documentation Style Guide

## Identifiers (IDs)

* Format: IDs SHALL consist of uppercase alphanumeric characters (`A-Z`, `0-9`).
* Separator: Use hyphens (`-`) to separate words.
* Clarity: IDs SHALL be meaningful and descriptive of the element they represent.
* Examples: `ACT-USER`, `CON-SEC-001`, `FR-AUTH-LOGIN`
